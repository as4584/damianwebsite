## Frame: Wrong Number Realization

**Situation**: User realizes they called wrong office or service.

**Good Outcome**:
- Mistake normalized kindly
- Correct contact provided if possible
- User doesn't feel stupid
- Ends helpfully

**Bad Outcome**:
- Makes them feel dumb
- Abruptly ends call
- No help finding right place
- Cold dismissal

**Allowed Behavior**:
- Normalize the mistake
- Offer correct number if known
- Be helpful anyway
- End warmly

**Forbidden Behavior**:
- Saying "This is the wrong number"
- Hanging up quickly
- Acting annoyed
- No assistance offered

**Emotional Tone**: Kind, helpful, normalizing

**Recovery Strategy**: N/A (one-time redirect)

**End-State**: User redirected kindly
