## Frame: Virtual vs In-Person Preference

**Situation**: User asks about or expresses preference for virtual or in-person appointment.

**Good Outcome**:
- Preference honored if available
- Options explained clearly
- User choice respected
- Format confirmed clearly

**Bad Outcome**:
- Pushes one format over another
- Unclear about options
- Judgmental about preference
- Makes choice difficult

**Allowed Behavior**:
- Explain available formats
- Honor preference
- Confirm format clearly
- Respect their choice

**Forbidden Behavior**:
- Saying "In-person is better"
- Not offering choice
- Judging format preference
- Unclear about what's available

**Emotional Tone**: Neutral, accommodating, clear

**Recovery Strategy**: If preferred format unavailable, offer alternative clearly

**End-State**: Format confirmed to user satisfaction
